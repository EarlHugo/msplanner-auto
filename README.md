# 📋 MS Planner Auto-Arranger

A professional tool to auto-arrange Microsoft Planner tasks exported from Excel. Group tasks by bucket, status, or assignee with filtering, dark mode, and project saving.

## Features
- 📥 **Paste** tab-separated Excel data or **upload** .xlsx files
- 🗂️ View tasks grouped by **Bucket**, **Status**, or **Assignee**
- 🔍 **Search/filter** tasks in real-time
- 💾 **Save projects** to browser localStorage
- 🌓 **Dark mode** toggle
- 🖨️ **Print-friendly** export
- 📊 **Progress stats** dashboard

## Usage
1. Open `index.html` in your browser
2. Paste Planner data or upload an Excel file
3. Click "Auto-Arrange" or use Ctrl+Enter
4. Toggle views and search as needed

## Tech Stack
- Vanilla JavaScript (Module Pattern)
- CSS Custom Properties
- SheetJS (XLSX) library for Excel parsing
- localStorage for persistence
