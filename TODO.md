# Host Website to GitHub Pages - TODO

## Steps
1. [x] Fix git divergence and push current `main` to origin
2. [x] Install GitHub CLI (`gh`) — downloaded to `gh_cli/bin/gh.exe`
3. [ ] Enable GitHub Pages on the repository
4. [ ] Verify deployment URL

## Manual Steps Required (Browser)
A GitHub Pages settings tab should already be open in your browser. If not, visit:
**https://github.com/EarlHugo/msplanner-auto/settings/pages**

Under **"Build and deployment"** > **"Source"**, select:
- **Deploy from a branch**
- Branch: `main`
- Folder: `/ (root)`
- Click **Save**

Your site will be live at:
**https://earlhugo.github.io/msplanner-auto/**

(Deployment takes 1–2 minutes after saving.)
